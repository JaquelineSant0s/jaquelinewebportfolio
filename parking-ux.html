Parking Lot System

This is the ux design project for the computerized parking system in Brazil. The system initially allows the registration of vehicles entering and leaving and calculating the parking value based on the variables pre-configured in the system, such as vehicle type, time of day and day of the week. The customer decided to create a computerized system using react Native programming language. I was hired to design the user experience from drafts of the system under construction and screenshots of competitors.

Customer Requirements

It is a system for pc environment, possibly on CRT monitors.

Has to accommodate content and settings based on an old version of the system and competitors.

There must be space for expansion of future features.

session for setting dates, times, events and holidays.

Price settings section.

Vehicle category configuration section.

price and categorie must be able to edit to be shown on the same screen.

Checkout session with manual editing.

user profile session.

This project was built in 4 weeks and two reviews were made before final delivery.
